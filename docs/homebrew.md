# Homebrew Release

This project is set up for Homebrew installs through a tap generated by GoReleaser.

## One-time setup

1. Push this repository to GitHub.
2. Create a tap repository named `homebrew-agent-audit`.
3. Create a GitHub token that can write to the tap repository.
4. Add that token to this repository as `HOMEBREW_TAP_GITHUB_TOKEN`.
5. Update the owner/name values in `.goreleaser.yaml` if the GitHub owner is not `sze`.

## Publish

Versions are managed by Release Please. Use Conventional Commit messages on `main`:

```text
feat: add codex adapter
fix: preserve PTY resize events
feat!: change database schema
```

Release Please opens a release PR with the changelog and version bump. Merge that PR to create the GitHub release and tag.

```sh
git push origin main
```

The tag then triggers the release workflow, which runs tests, builds release archives, publishes checksums, and writes a Homebrew formula to the tap.

## Install

```sh
brew tap sze/agent-audit
brew install agent-audit
```

If the tap owner is different, replace `sze` in the commands above.
